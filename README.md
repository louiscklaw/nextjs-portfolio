# README.md

### this repo hold the source of portfolio.louislabs.com

### spin me up
```
cd <PROJECT ROOT>
yarn
```

### deploy to staging
`yarn deploy_staging`

### deploy to production
`yarn deploy_production`

### to test
`yarn test`
